larochelle(Qair)R Documentation

contient des donnees de pollution sur la rochelle. (France)

Description

Les donnees concernees ont ete mesurees en 2007 sur les 3 stations d'Atmo Poitou-Charentes.

Usage

data(larochelle)

Format

A data frame with 35040 observations on the following 8 variables.

date
a chron - les dates des mesures, en quart-horaire
O3_VER
a numeric vector - concentration d'ozone sur la station de Verdun.
N2_VER
a numeric vector - concentration de dioxyde d'azote sur la station de Verdun.
NO_VER
a numeric vector - concentration de monoxyde d'azote sur la station de Verdun.
O3_VAU
a numeric vector - concentration d'ozone sur la station de Vaugoin.
O3_AYT
a numeric vector - concentration d'ozone sur la station d'Aytre.
DV_ATM
a numeric vector - direction de vents au siege d'Atmo Poitou-Charentes.
VV_ATM
a numeric vector - vitesse de vents au siege d'Atmo Poitou-Charentes.

Source

http://www.atmo-poitou-charentes.org

Examples

data(larochelle)
plot(larochelle["07-01-01", "07-01-10"][grep("O3", names(larochelle), value=TRUE)], ask.leg=FALSE)
