mesures(Qair)R Documentation

Acces aus attributs d'objets 'Qair' et/ou 'rose'.

Description

Permet dd'acceder aux informations relatives aux donnees contenues dans des objets de type 'Qair' et 'rose'. (mesures, unites, stations)

Usage

  mesures(donnees)
  ## S3 method for class 'Qair':
  mesures(donnees)
  ## S3 method for class 'rose':
  mesures(donnees)

  stations(donnees)
  ## S3 method for class 'Qair':
  stations(donnees)
  ## S3 method for class 'rose':
  stations(donnees)

  unites(donnees)
  ## S3 method for class 'Qair':
  unites(donnees)
  ## S3 method for class 'rose':
  unites(donnees)

  ## S3 method for class 'Qair':
  names(x)
  ## S3 method for class 'rose':
  names(x)

Arguments

x Un objet de la classe 'Qair' ou un objet de la classe 'rose'.
donnees Un objet de la classe 'Qair' ou un objet de la classe 'rose'.

Value

Renvoie un vecteur de 'character' decrivant les mesures/stations/unites/noms correspondant aux donnees contenues dans 'donnees' ou 'rose'.

Note

Un element de classe 'rose' aura les memes mesures/stations/unites/noms que l'element 'Qair' dont il est issu.

See Also

[.Qair

Examples

  data(larochelle)
  mesures(larochelle)
  stations(larochelle)
  unites(larochelle)
  names(larochelle)

  rose <- calcRose(larochelle, ~DV_ATM+VV_ATM)
  mesures(rose)
  stations(rose)
  unites(rose)
  names(rose)
