as.list.rose(Qair)R Documentation

Transformation d'un objet 'rose' en objet 'list'

Description

Permet d'obtenir une list a partir d'un objet de la classe rose

Usage

## S3 method for class 'rose':
as.list(x, ...)

Arguments

x un objet de la classe 'rose' a convertir en list.
... Arguments to pass to or from other methods

Value

La fonction renvoie un element de type 'list' dont les sous-elements sont du type 'Qair'.

See Also

calcRose

Examples

  data(larochelle)
  rose <- calcRose(larochelle, ~DV_ATM+VV_ATM)
  summary(as.list(rose))
