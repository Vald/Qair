intervalles(Qair)R Documentation

Acces aux intervalles d'une rose

Description

Permet d'acceder aux intervalles definissant les secteurs d'un objet 'rose'.

Usage

intervalles(rose)
## S3 method for class 'rose':
intervalles(rose)

Arguments

rose Un objet de type 'rose', agggrege ou non.

Value

Renvoie une liste de vecteurs. Chacun de ces vecteurs definit les bornes inferieure et superieure d'un secteur de vent.

See Also

calcRose, plot.rose

Examples

  data(larochelle)
  intervalles(calcRose(larochelle, ~DV_ATM+VV_ATM))
